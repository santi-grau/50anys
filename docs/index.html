This is an interactive logo to commemorate the 50th Anniversary of Eina's Design Foundation in Barcelona:

1. Every day at 00h 00m 00s a new logo will be generated consisting in a set of blocks assorted randomly. Each block will contain a texture taht may ( or may not ) be animated.

2. Anytime during the day anyone can visit the interactive version of the logo ( currently located at http://proper-code.com/50anys ) and modify the logo's composition. Either changing the textures in each block ( by clicking on them ) or by deciding if this textures must be animated or not ( by long clicking on the blocks ).

3. Every day, at 23h 59m 59s, the logo will be saved and a new one will be generated.

4. Previously generated logos will be stored and availabe online in order to be able to see their evolution. They can be visited by clicking prev and next ( or swiping on a mobile ).

5. Each logo version can be exported for print clicking the top-right button ( desktop only )

The goals of this experience are: 1. Make design a col.laborative experience at everyone's reach 2. Create an evolutionary logo that expresses and shows how time passes 3. Commemorate Eina's 50 year anniversary emphasizing it's diversity and plurality.

Logotip interactiu pel 50 aniversari de la Fundació Eina::

1. Cada dia a les 00h 00m 00s es genera un nou logotip mitjançant una sèrie de blocs distribuïts de forma aleatòria. Cada bloc conté una textura que pot ser, (o no), animada.

2. Durant la resta del dia tothom pot visitar la versió interactiva del logo i modificar la seva composició. Ja sigui modificant la textura de cada bloc ( fent click en els blocs ) o decidint si les textures son animades o no ( fent 'clic llarg' en els blocs ).

3. Al final del dia, a les 23h 59m 59s, el logotip es guardarà i es generarà un de nou.

4. Els logos generats quedaran enmagatzemats i es podran visitar per tal de veure la seva evolució. Es pot navegar pels logos clicant als botons anterior / següent ( o fent swipe al mòbil ).

5. Totes les versions del logo es poden exportar per imprimir fent click al boto de dalt a la dreta ( només versio desktop ).

Els objectius d'aquesta experiència són: 1. Fer del disseny una experiència col.laborativa, a l’abast de tothom. 2. Fer un logo evolutiu que mostri el pas del temps. 3. Enfatitzar la diversitat i pluralitat que ha fet d’Eina una escola de referència.
